My HTML Cheat Sheet

Headings

Headings from h1 to h6 identify blocks and sections. Every page must have one h1.

Paragraphs

Paragraphs of text go in p tags.

Lists

Unordered Lists

Unordered lists are marked with bullet points and go in ul tags. Each list item goes in a li tag.

Types of lists:

  • Unordered lists ul
  • Ordered lists ol

Ordered Lists

Ordered lists are numbered and go in ol tags. Each list item goes in a li tag.

Polya's Problem Solving Framework

  1. Understand the problem
  2. Come up with a plan
  3. Carry out the plan
  4. Go back and improve your solution

Anchors (i.e. links)

Anchor tags, a, create links to other pages. The URL goes in the href attribute. Go to DuckDuckGo to search for more information.

Setting target="_blank" will open the link in a new window. See MDN for more details.

Images

You can use the img tag to add images to your page.

A table is a structured set of data made up of rows and columns (tabular data). A table allows you to quickly and easily look up values that indicate some kind of connection between different types of data, for example a person and their age, or a day of the week, or the timetables for a local swimming pool.
